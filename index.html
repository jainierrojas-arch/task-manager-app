Programacion de contenido (Make.com webhook)
No configurado

Como configurar Make.com para programar en Instagram:

  1. Crea un escenario en Make.com
  2. Modulo trigger: Webhooks → Custom webhook
  3. Make te da una URL — pegala arriba y guarda
  4. Modulo siguiente: Instagram for Business → Create a Photo Post (o Reel/Story)
  5. Mapea los campos del webhook: caption, mediaUrl, scheduledAt
  6. Activa el escenario. Ahora cada vez que programes algo desde la app, Make lo publica automaticamente.
Programacion en TikTok (GoHighLevel webhook)
No configurado

Como configurar GHL para postear en TikTok:

  1. En GHL: Settings → Social Planner → Connect Accounts → TikTok. Login con cuenta TikTok Business.
  2. Automation → Workflows → New Workflow
  3. Trigger: Inbound Webhook → copiá la URL que GHL te genera y pegala arriba.
  4. Action: Social Planner → Create Social Post. Mapeá caption y mediaUrl desde el payload del webhook. Tildá tu cuenta TikTok como destino.
  5. Activa el Workflow. Cuando programes con TikTok marcado, GHL publica.
Cloudinary (subir imagenes y videos directo desde la app)
No configurado

Como configurar Cloudinary (5 minutos):

  1. Login a cloudinary.com → Settings → Upload → Upload Presets
  2. Click Add upload preset
  3. Name: task-manager-uploads (o el que quieras)
  4. Signing mode: Unsigned
  5. Folder: task-manager (opcional, separa tus uploads)
  6. Allowed formats: jpg, png, mp4, mov, jpeg, webp
  7. Max file size: 100 MB (limite de IG para reels)
  8. Save
  9. Pega arriba el Cloud Name (lo ves en el dashboard, ej. dcdfsfjha) y el Upload Preset name que creaste, y guarda. Listo.

Despues de guardar: en el modal Programar veras un boton 📁 Subir archivo al lado del campo de URL. Click, eliges una imagen o video local, se sube automaticamente a Cloudinary y la URL se rellena sola.

🎤 Transcripción de videos (OpenAI Whisper)
No configurado

Cómo configurar Whisper (2 minutos):

  1. Andá a platform.openai.com/api-keys y creá una API key
  2. Pegala arriba y guardá
  3. En el Depósito, los videos van a tener un botón 🎤 Transcribir que extrae el texto del audio. Costo: ~$0.006/min de video (re-barato).
  4. Una vez transcrito, podés clickear ✨ Recrear guion para que Claude reescriba el guion con un ángulo distinto.

API key se guarda por workspace — si tenés Mi Agencia y Cliente Pizza, podés tener API keys diferentes (o compartir la misma).

🔀 Admin: Migrar contenido entre workspaces
→

Soluciona "no le aparece a los chicos lo que yo subo". Cada workspace está aislado — si vos estás en "Agencia" y los chicos en "Prueba de Cliente", no ven lo que vos creás. Esta herramienta mueve TODO (entries del depósito, categorías, tareas, proyectos, programaciones) del workspace source al target.

Pasos: 1) Elegí el workspace de origen (de donde sacar el contenido). 2) Elegí el de destino. 3) Click "Migrar". 4) Confirmá. 5) Decíles a los chicos que cambien al workspace destino (selector arriba a la izquierda).
